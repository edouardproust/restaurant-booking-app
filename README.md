# Restaurant booking web app

## Description

A web application that allows users to **book a table online** at a the restaurant Little Lemon, Chicago. This project was completed as part of the **Meta Front-End Developer Professional Certificate**. The app is fully built with **React**, focusing on front-end development, UI design, and state management.

![image](https://user-images.githubusercontent.com/45925914/213962562-eb57c8d6-be50-4202-9f87-abdda06df9b1.png)

## Features

* Online table booking with interactive form
* Front-end management of table availability
* Responsive and user-friendly interface

## Technologies Used

* **React** (components, hooks, state management)
* **JavaScript** (ES6+)
* **CSS** for styling

## Installation & Running

1. Clone the repository:

   ```bash
   git clone https://github.com/edouardproust/restaurant-booking-app.git
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the app in development mode:

   ```bash
   npm start
   ```
4. Build for production:

   ```bash
   npm build
   ```
